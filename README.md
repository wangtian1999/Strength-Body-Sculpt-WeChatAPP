# 健力塑身 (Strength & Body Sculpt)

一款为 0~2 年健身初学者打造的极简、硬核、理工风微信小程序。

## 核心理念
**先预估身材 → 定期望目标 → 按身材推算力量 → 自动生成渐进训练计划**

我们拒绝臃肿的社交功能、拒绝无尽的广告干扰、拒绝繁琐的饮食打卡。这里只有纯粹的算法工具，帮助你通过科学的力量增长来驱动身材的蜕变。

## 核心功能

### 1. 智能身材录入 (Body Input)
- **双模式切换**：支持“极简范围选择”与“精细数据录入”。
- **即时算法**：自动计算 BMI、BMR、TDEE 及瘦体重 (LBM)。
- **视觉参考**：内置男女体脂参考图，辅助小白准确判断当前身形。

### 2. 目标身材规划 (Target Planning)
- **体脂驱动**：基于理想体脂率自动推算目标体重。
- **动态预估**：推算到达目标所需的周数及每日热量盈余/缺口。
- **营养建议**：提供硬核的三大营养素（蛋白质、碳水、脂肪）克数配比。

### 3. 力量推算与评级 (Strength & Standards)
- **三大项预估**：基于身材数据自动推算深蹲、卧推、硬拉的 1RM 水平。
- **力量等级**：实时判定你的力量处于“未入门、新手、初级、精英”哪个阶段。
- **Wilks 分数**：引入国际通用系数，客观衡量力量水平。
- **起步修正**：支持手动微调 5x5 起步重量，并提供“空杆测试”安全引导。

### 4. 自适应训练计划 (Adaptive Plan) - v5.0
- **智能生成**：一键生成 12 周线性渐进超负荷计划。
- **精力管理**：引入 Energy Score（精力评分），每日根据睡眠与压力动态调整训练容量。
- **微量进阶**：根据年龄与 BMI 自动匹配加重步进（0.5kg - 2.5kg），侧重长期稳健增长。
- **动作详情**：内置动作要领、退阶与进阶建议，解决“动作不标准”与“做不动”的困扰。
- **疲劳管理**：内置减载周（Deload）诊断逻辑，自动识别平台期并提供恢复方案。
- **补给指南**：将宏量营养素换算为具体的食物量（支持肉类、素食、高性价比等多种模式）。

## 产品气质
- **极简**：Apple Style 列表设计，无圆角、无冗余装饰，移除所有繁琐括号。
- **硬核**：理工男审美，只谈数据与算法，强调“力量驱动身材”。
- **干净**：无登录、无广告、无社区、全本地存储。

## V5.0 自适应升级说明
在 v5.0 版本中，我们引入了更加严谨的安全性阈值：
1. **BMI 风险对冲**：针对 BMI > 28 的用户，系统会自动下调推算力量，优先保护关节。
2. **高龄友好逻辑**：针对 50 岁以上用户，加重步进自动减半，延长线性收益期。
3. **术语透明化**：在“我的”页面增加了 TDEE、LBM、LP 等核心名词的点击解释。
