# 健力塑身 (Strength & Body Sculpt)

一款为 0~2 年健身初学者打造的极简、硬核、理工风微信小程序。

## 核心理念
**先预估身材 → 定期望目标 → 按身材推算力量 → 自动生成渐进训练计划**

我们拒绝臃肿的社交功能、拒绝无尽的广告干扰、拒绝繁琐的饮食打卡。这里只有纯粹的算法工具，帮助你通过科学的力量增长来驱动身材的蜕变。

## 核心功能

### 1. 智能身材录入 (Body Input)
- **双模式切换**：支持“极简范围选择”与“精细数据录入”。
- **即时算法**：自动计算 BMI、BMR、TDEE 及瘦体重 (LBM)。
- **视觉参考**：内置男女体脂参考图，辅助小白准确判断当前身形。

### 2. 目标身材规划 (Target Planning)
- **体脂驱动**：基于理想体脂率自动推算目标体重。
- **动态预估**：推算到达目标所需的周数及每日热量盈余/缺口。
- **营养建议**：提供硬核的三大营养素（蛋白质、碳水、脂肪）克数配比。

### 3. 力量推算与评级 (Strength & Standards)
- **三大项预估**：基于身材数据自动推算深蹲、卧推、硬拉的 1RM 水平。
- **力量等级**：实时判定你的力量处于“未入门、新手、初级、精英”哪个阶段。
- **Wilks 分数**：引入国际通用系数，客观衡量力量水平。
- **起步修正**：支持手动微调 5x5 起步重量，并提供安全引导。

### 4. 健力塑身训练计划 (SBD Plan)
- **多维分化**：支持“全身分化 (A/B)”、“二分化 (上肢/下肢)”及“三分化 (推/拉/腿)”三种经典模式，满足不同进阶需求。
- **智能生成**：一键生成 12 周线性渐进超负荷计划，专注于深蹲、卧推、硬拉三大项。
- **频率建议**：明确标注“练一休一”及每周 3 次的科学训练频率。
- **微量进阶**：根据年龄与 BMI 自动匹配加重步进（0.5kg - 2.5kg），确保长期稳定进步。
- **动作详情**：内置动作要领、退阶与进阶建议，确保动作质量。
- **极简追踪**：Apple Style 任务清单，一键勾选完成，实时计算 12 周进化矩阵。

## 产品气质
- **专业**：聚焦“力量驱动身材”核心逻辑，提供多种分化方案，回归训练本质。
- **极简**：Apple Style 列表设计，界面干净利落。
- **硬核**：用数据说话，量化每一分进步。

## V5.0 专业化升级说明
在 v5.0 版本中，我们进行了精简化重构，使其更聚焦于核心功能：
1. **训练分化升级**：增加了二分化和三分化计划，用户可根据自身恢复能力和时间安排自由选择。
2. **动态 5x5 方案**：不再死板地维持 5x5。系统会根据用户等级（新手、中级、高级）动态调整核心项的组数与次数（如高级用户切换为 5x3 纯力量模式），确保训练负荷与恢复能力的精准匹配。
3. **起步重量智能建议**：根据训练年限自动调整起步百分比（50%-80%），新手预留更多进步空间，老手则直接进入高强度对抗。
4. **剔除冗余**：移除了今日状态调整、饮食参考、减载周等可有可无的功能，使应用更精准。
5. **BMI 风险对冲**：针对 BMI > 28 的用户，自动下调起步重量，保护关节。
6. **高龄友好逻辑**：针对 50 岁以上用户，加重速度自动减半。
7. **频率引导**：在计划页面增加了每周 3 次、练一休一的明确建议。
